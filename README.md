- 🛡️ **29+ категорий** черных списков для комплексной защиты
- 🔄 **Регулярные обновления** для актуальной защиты
- 📁 **Модульная структура** — используйте только нужные категории
- 🌍 **Открытый исходный код** — полная прозрачность

1. Скачайте нужные файлы из папки `/blacklist`
